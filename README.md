## نظرة عامة

صُممت هذه الأداة خصيصاً للتقسيم الإداري السوري، لحل مشكلة شائعة: لديك قائمة مواقع (فروع، مستودعات، عملاء، مشاريع...) بإحداثيات جغرافية فقط، وتحتاج معرفة **أي محافظة ومنطقة وناحية** تقع فيها كل نقطة — دون قضاء ساعات في المطابقة اليدوية.

| بدون الأداة | مع الأداة |
|---|---|
| مطابقة يدوية بطيئة وعرضة للخطأ | مطابقة تلقائية فورية لآلاف المواقع |
| لا رؤية جغرافية للتوزيع | خريطة تفاعلية + إحصائيات فورية |
| تصدير غير منظم | تصدير Excel مخصّص (أعمدة + لغة) |

## المميزات

| الميزة | الوصف |
|---|---|
| **رفع ذكي** | سحب وإفلات لملف Excel، مع كشف تلقائي لأعمدة الإحداثيات وإمكانية تصحيحها يدوياً |
| **مطابقة دقيقة 100٪** | كل موقع يُطابَق مع أقرب نقطة فعلية ضمن جدول قياسي يضم أكثر من 8000 قرية وحي في سوريا |
| **جدول تفاعلي** | بحث وترقيم صفحات، يعرض أعمدة ملفك الأصلية + الأعمدة الإدارية المستخرجة جنباً إلى جنب |
| **خريطة حية** | عرض على مستوى محافظة / منطقة / ناحية، مع إمكانية الضغط على أي منطقة لتصفية النتائج بها مباشرة |
| **تصفية متتالية** | محافظة ← منطقة ← ناحية، تنعكس فوراً على الجدول والخريطة والإحصائيات والتصدير |
| **إحصائيات فورية** | إجمالي المواقع وتوزيعها التفصيلي على كل مستوى إداري |
| **تصدير مرن** | اختيار الأعمدة المطلوبة، واللغة (عربي / إنجليزي)، مع دعم اتجاه RTL في ملف الإخراج |
| **وضعان للعرض** | داكن وفاتح، يُحفظ تفضيلك تلقائياً في المتصفح |
| **خصوصية كاملة** | كل المعالجة تتم محلياً داخل متصفحك — لا تُرسل أي بيانات لأي خادم خارجي |

## آلية العمل

```
  1. رفع الملف   →   2. تأكيد الأعمدة   →   3. مطابقة أقرب نقطة    →   4. عرض وتصدير
     Excel              (خط الطول/العرض)      (~8007 نقطة قياسية)        جدول / خريطة / إحصائيات
```

1. **الرفع** — يُقرأ ملف الإكسل داخل المتصفح مباشرة دون رفعه لأي خادم.
2. **تحديد الأعمدة** — كشف تلقائي لعمودي خط الطول وخط العرض، مع معاينة للتأكيد.
3. **المطابقة** — تُحسب المسافة الجغرافية الدقيقة (معادلة Haversine) بين كل موقع مرفوع وجميع النقاط في الجدول القياسي، وتُعتمد بيانات **أقرب نقطة فعلياً** (وليست تقريباً) بفضل فهرسة جغرافية تضمن نفس دقة الفحص الشامل مع أداء أسرع بكثير.
4. **الحدود الجغرافية** (GeoJSON) تُستخدم فقط لرسم الخريطة، وليست جزءاً من آلية التصنيف نفسها.
5. **العرض والتصدير** — الجدول والخريطة والإحصائيات وملف Excel المُصدَّر جميعها مبنية من نفس البيانات المطابقة، وتتفاعل مع عوامل التصفية لحظياً.

## طريقة الاستخدام

| الخطوة | الإجراء |
|:---:|---|
| **1** | افتح [`index.html`](index.html) بالنقر المزدوج — يعمل مباشرة في أي متصفح |
| **2** | ارفع ملف الإكسل الخاص بمواقعك، أو اضغط "تجربة بيانات توضيحية" للتجربة السريعة |
| **3** | تأكد من عمودي خط الطول/العرض المكتشفين تلقائياً، ثم اضغط "مطابقة البيانات" |
| **4** | استعرض النتائج، وصفِّها حسب المحافظة/المنطقة/الناحية عند الحاجة |
| **5** | من قسم "تصدير النتائج": اختر الأعمدة واللغة، ثم "تنزيل ملف Excel" |

> **متطلبات ملف الرفع:** عمود واحد لخط الطول (Longitude) وعمود واحد لخط العرض (Latitude) على الأقل. أي أعمدة إضافية (اسم الموقع، النوع، ...) تُحفظ وتظهر كما هي في النتائج والتصدير.

## مصادر البيانات

الجدول القياسي وحدود المحافظات/المناطق/النواحي مبنية على تصنيف التقسيم الإداري السوري المعتمد من الأمم المتحدة **(OCHA — Common Operational Datasets)**، بأربعة مستويات:

`محافظة (Governorate)` ← `منطقة (District)` ← `ناحية (Subdistrict)` ← `قرية / حي (Community / Neighborhood)`
